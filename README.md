# Adelvo

**Professional software tools for live video production, audio routing, and media workflows.**

🌐 [adelvo.io](https://adelvo.io)

## Products

### 🎬 [Directors Plan](https://adelvo.io/directors-plan)
Professional vMix control & automation software for live productions. Timeline-based rundown planning, drag & drop commands, Stream Deck integration, 19+ languages. For churches, sports broadcasts, and corporate events.

### 🎚️ [MixDesk X](https://github.com/adelvo-software/MixDeskX) &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;![Free](https://img.shields.io/badge/Free-3b82f6?style=flat) ![Windows](https://img.shields.io/badge/Windows-10b981?style=flat) ![macOS](https://img.shields.io/badge/macOS-ff7e00?style=flat)
Free multi-device audio mixer & router. CLAP effect chains with native plugin GUIs on every input and bus, 2/3/11-band EQ, per-bus delay to sync Bluetooth with wired speakers, 9 UI languages, in-app updates. [Download →](https://github.com/adelvo-software/MixDeskX/releases/latest)

![MixDesk X](images/mixdeskx.jpg)

### 🔊 [Volume Balance](https://adelvo.io/volume-balance)
Browser extension that normalizes audio levels across all tabs. Audio compressor with threshold, ratio, knee, attack, release, and makeup gain. Chrome, Firefox, Safari.

### 🎛️ [MacMixDesk](https://adelvo.io/macmixdesk)
Professional audio routing for macOS. Capture individual app audio, mic, webcam, and line-in simultaneously. Up to 4 stereo outputs with EQ.

## Free Tools

### 🎬 [DP Thumbnail Server](https://github.com/adelvo-software/dp-thumbnail-server)
Fast vMix input thumbnail generator. Replaces vmix-snapshot-proxy with support for all input types, key-based URLs, and a web UI. [Download →](https://adelvo.io/directors-plan/#thumbnail-server)
